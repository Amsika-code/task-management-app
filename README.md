# Task Management App

A clean, lightweight task management web app built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

## Live Demo

🔗 [View Live](https://amsika-code.github.io/task-management-app)

---

## Features

- Add new tasks with a single click or by pressing Enter
- Mark tasks as complete
- Delete tasks individually
- Minimal, responsive UI
- Zero dependencies — pure HTML, CSS, and JavaScript

---

## Tech Stack

| Technology | Purpose |
|------------|---------|
| HTML5 | Structure and markup |
| CSS3 | Styling and layout |
| JavaScript (Vanilla) | DOM manipulation and task logic |

---

## Project Structure

```
task-management-app/
├── index.html     # Main HTML structure
├── style.css      # Styling
└── script.js      # Task logic (add, complete, delete)
```

## Getting Started

No installation needed. Just open the project locally:

```bash
# Clone the repository
git clone https://github.com/Amsika-code/task-management-app.git

# Open in browser
cd task-management-app
open index.html
```

Or simply visit the [live demo](https://codepen.io/editor/amsika/pen/019e4093-1d85-7e3d-a232-66bff123e0c4).

---

## How It Works

1. Type a task into the input field
2. Click **Add** to add it to your list
3. Click a task to mark it complete
4. Delete tasks when you're done
